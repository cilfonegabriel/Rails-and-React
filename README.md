## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running follow these simple example steps.

### Prerequisites

you have to those tools in your local machine.

- [ ] Ruby
- [ ] Rails
- [ ] PostgresSQL
- [ ] GIT & GITHUB
- [ ] Any Code Editor (VS Code, Brackets, etc)

### Clone Repo

Clone the project using the following bash command in an appropriate location.

bash
  git clone https://github.com/cilfonegabriel/Rails-and-React.git


Go to the project directory.

bash
  cd Rails-and-React


### Add necessary packages

For installing necessary packages, run the following bash command:

```ruby
  bundle install
```
### Run the server in development mode

In the project directory, you can run the project by using following command:

```ruby
  rails server
```
Open a new terminal and run the next command to start react:

```
yarn build --watch
```

And now you can visit the site with the URL http://localhost:3000
